# Amazon Sales Analysis

## Overview
This project analyzes Amazon sales data using Python to identify trends in revenue, discounts, product performance, and customer behavior.

## Objectives
- Explore the structure and quality of the dataset
- Analyze sales patterns and revenue drivers
- Examine the relationship between discounts, ratings, and sales
- Build a regression model to support business insights

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Files
- `MDA760_Amazon_Sales.ipynb` — main notebook containing data cleaning, analysis, visualizations, and modeling

## Key Insights
- Discounts influenced purchasing behavior and total revenue
- Customer ratings and review counts helped explain product performance
- Visual analysis highlighted category-level and sales-pattern differences
- Regression modeling was used to study relationships between variables

## Outcome
This project demonstrates skills in exploratory data analysis, visualization, and business-focused interpretation of sales data using Python.
